# FUTURE_CS_02  
Cyber Security Internship – Task 2  
Phishing Email Detection & Awareness System  

## Objective  
This project focuses on analyzing phishing email samples, identifying phishing indicators, classifying risk, and creating user awareness guidelines to prevent phishing attacks.

## Tools Used  
- VirusTotal (URL Reputation Analysis)  
- WHOIS Lookup (Domain Investigation)  
- Email Content Analysis  
- Browser-based Investigation Tools  

## Analysis Summary  
A phishing email containing urgent language and a suspicious HTTP link was analyzed. 
The domain showed low reputation and redacted ownership information. 
The email was classified as **High Risk Phishing**.

## Key Phishing Indicators  
- Generic greeting  
- Urgent / fear-based message  
- Suspicious link (HTTP)  
- No official sender identity  

## Awareness Outcome  
The report provides prevention guidelines and employee awareness measures to help identify and avoid phishing attacks.
